# IE7 Browser Emulator (PyQt6)

Небольшой эмулятор **Windows Internet Explorer 7**, написанный на Python и PyQt6 с использованием Qt WebEngine.  
Цель проекта — ностальгичный браузер с интерфейсом, похожим на IE7: тулбары, кнопки «Назад/Вперёд», избранное, журнал, менеджер загрузок и т.п.

## Возможности

- Просмотр веб‑страниц на движке Qt WebEngine.
- Кнопки навигации «Назад» и «Вперёд» в стиле IE7.
- Домашняя страница (по умолчанию: https://www.google.com).
- Журнал посещений с поиском.
- Избранное с отдельным окном управления.
- Менеджер загрузок с отображением размера и времени.
- Простое окно «Сертификаты безопасности».
- Скриншоты текущей страницы.
- Автономный режим (имитация offline).
- Сохранение истории и избранного в `ie7_settings.json`.

## Требования

- Python 3.10+ (проверено на Windows).
- PyQt6
- PyQt6-WebEngine

Установить зависимости:

```bash
pip install -r requirements.txt
```

## Запуск

Из папки проекта:

```bash
python NewPyIE7.py
```

При первом запуске создаётся папка для загрузок, обычно:

```text
C:\Users\<Имя пользователя>\IE7 Downloads
```

а рядом с `NewPyIE7.py` — файл настроек `ie7_settings.json`.

## Структура проекта

- `NewPyIE7.py` — основной файл приложения (окно браузера, диалоги и логика).
- `ie7_back.png`, `ie7_forward.png` — иконки кнопок «Назад» и «Вперёд» на тулбаре.
- `requirements.txt` — список зависимостей.
- `README.md` — описание проекта.

## Известные ограничения

- Проект является эмулятором, а не полноценным браузером.
- «Инструменты разработчика» и работа с реальными SSL‑сертификатами не реализованы, только заглушки.
- Поведение может отличаться от оригинального IE7.

## Лицензия

Проект создан в образовательных и ностальгических целях и не связан с Microsoft.
